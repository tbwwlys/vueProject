- react 和 vue 的区别

    - vue 好入门 template script style 指令 事件 都好用
    - react 全面函数式编程
        JSX 模板语法 编译阶段编译， js 里面写html
    - vue 支持双向/单向绑定  react 只支持 单项绑定
        双向绑定 性能不好
    - react 没有指令 v-if v-for
        事件模拟原生事件
    - 全面函数式编程拥抱 hooks 
        useState 
    - ts 支持完美
        React.FC<>      将函数渲染成到页面
    - vue 内置css 
        react 引入的，css 独立文件
